# NestJS practise

## Description

NestJS practise app.

## Installation

```bash
# setup
$ docker-compose up -d
$ yarn install
$ yarn db:migrate-generate
```

## Running the app

```bash
# development
$ yarn local
```

## Test

```bash
# unit tests
$ yarn test:unit

# e2e tests
$ yarn test:e2e-parallel
```
